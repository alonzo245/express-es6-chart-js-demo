## 🎈 Usage
---
```
1. git clone https://github.com/alonzo245/express-es6-chart-js-demo.git
```
```
2. cd backend/
```
```
3. npm i
```
```
4. In dev mode - npm run watch
```
```
5. In prod mode - npm run build && npm run prod
```
```
6. Open your browser on http://localhost:3000/
```
